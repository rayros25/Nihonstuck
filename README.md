The Nihonstuck team is working to bring the internet's most beloved webcomic, [Homestuck](https://en.wikipedia.org/wiki/Homestuck), to 
a whole new audience.

This is a mod for the Unofficial Homestuck Collection (UHC), which can be found [here](https://homestuck.github.io/).
Alterniatively, the translation is also hosted on [our website](https://nihonstuck.github.io), albeit with a few inconsistencies.

## Install

First install the Unofficial Homestuck Collection, linked above.
Then clone this repo into the `mods` folder, in the same place you store your Asset Pack.
Finally, in Settings, scroll down to Mod Settings and drag the Nihonstuck mod from Inactive to Active.
